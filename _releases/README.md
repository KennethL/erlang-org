# Blogs posts

News items are displayed under `/blogs` and `/blog/{{title}/`.

Use the code below as a template:

```markdown
---
id: 140
layout: post
title: "My new news item"
lead: "The lead shown on index.html"
date: "2020-05-13"
author: "John Doe"
---
Some markdown

## Sub title

Some more markdown
```
